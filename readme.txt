== Description ==

Easily integrate Calq.io Analytics into your WordPress website.

Calq.io is an analytics service for tracking visitors and much more on your websites and your Apps.

This plugin offers a simple interface to adding the necessary Calq.io Javascript to your front-end pages, and a few options for tweaking the Javascript.

== Frequently Asked Questions ==

= I'm having problems with Calq.io and statistics etc.? =

We are not Calq. We have only created the WordPress plugin to help ourselves, and then shared it with you.
Please contact Calq directly for issues and questions relating to their service.

= What are the web hosting requirements for this plugin? =

This plugin requires that your web hosting and your WordPress site meet certain basic requirements. These will be checked before fully loading the plugin.

* PHP Version 5.3.0 or greater. Check with your web host why on earth you are running on less than this.
* WordPress Version 3.5.0 or greater.  You should ideally run the absolute latest, so if you have problems deactivate this plugin.

= Is this plugin made by Calq.io? =

No. It's made by [iControlWP](http://www.icontrolwp.com)

= Is Calq.io Free? =

Yes, up to a certain number of users/visitors.

= What is Calq.io? =

[Calq.io](http://calq.io) is an analytics/tracking solution. A bit like Google Analytics, Kissmetrics, Mixpanel etc.
